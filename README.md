Conversor de Dólar para Real

Um projeto simples desenvolvido com HTML, CSS e JavaScript que permite converter valores de dólar para real.

Funcionalidades:

- Converter valores de dólar para real
- Interface simples e fácil de usar
- Cálculo instantâneo ao inserir o valor

Tecnologias utilizadas:

- HTML5
- CSS3
- JavaScript

Como usar:

1. Digite o valor em dólar
2. Clique no botão de converter
3. O valor em real será exibido na tela

!! A cotaçao do Dolar utilizada neste projeto é fixa de 5 reais (Valor do dolar no dia 20 de abril de 2026).
